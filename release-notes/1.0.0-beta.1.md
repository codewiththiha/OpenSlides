## What's New

- First public beta of OpenSlides.
- Create local-first code slide decks with Shiki syntax highlighting and stepped highlights.
- Present with Magic Move transitions, slide search, autoplay, and light or dark themes.

## Full changelog

- Local-first project and slide management backed by SQLite.
- Shared-worker Shiki syntax highlighting with lazy language and theme loading.
- Stepped highlights with Magic Move transitions and presentation autoplay.
- FTS5 slide search, lazy thumbnails with hover previews, and resizable panels.
- Per-slide undo and redo plus an exactly-once quit handshake that flushes pending edits.
